## Implementation Steps

### Step 1: Clone Locally

`clasp clone 1ABC...XYZ` to pull all files. Set up `.claspignore`.

### Step 2: Refactor inventory.gs

Batch items into chunks of 50, POST each chunk to v2 API endpoint:
```javascript
function processInbound(items) {
  var chunks = chunkArray(items, 50);
  var results = [];
  for (var i = 0; i < chunks.length; i++) {
    var response = UrlFetchApp.fetch(CONFIG.API_V2_URL, {
      method: 'post',
      payload: JSON.stringify({ items: chunks[i] }),
      contentType: 'application/json'
    });
    results.push.apply(results, JSON.parse(response.getContentText()).data);
  }
  return results;
}
```

### Step 3: Add Dashboard Formatting

`formatDashboard()`: bold header row, freeze row 1, auto-resize columns.

### Step 4: Update Config

Set `API_V2_URL`, `BATCH_SIZE`, `DASHBOARD_SHEET`, `LOG_SHEET` constants.

### Step 5: Add Utility

`chunkArray(array, size)` — slice array into groups of `size`.

### Step 6: Update Manifest

Add `https://www.googleapis.com/auth/drive.readonly` scope.

### Step 7: Push and Test

Run `clasp push` to upload all modified files. Open spreadsheet and test
`processInbound` with sample data. Verify dashboard formatting.
