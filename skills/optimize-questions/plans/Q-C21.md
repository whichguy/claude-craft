# Plan: CSV Data Import Tool for Analytics Platform

## Context

Analytics team needs to import partner CSV data files. Files range from small (100 rows)
to very large (multi-GB). Upload via REST endpoint, validate against schema, insert
valid records into `partner_data` table. Database has ~50M existing rows.

## Current Architecture

- Express.js API, PostgreSQL analytics database
- Manual SQL imports, no upload capability

## Implementation Steps

### Step 1: Upload Endpoint

Multer-based file upload to `/tmp/uploads`. Auth middleware required.

### Step 2: Import Handler

```typescript
export async function csvImportHandler(req: Request, res: Response) {
  const batchId = uuidv4();

  // Read the entire uploaded file into memory
  const fileContent = fs.readFileSync(req.file.path, 'utf8');

  // Parse all rows at once (synchronous)
  const rows = csvProcessor.parseAll(fileContent);

  // Validate all rows
  const validated = rows.map(row => rowValidator.validate(row, req.body.partnerId));
  const valid = validated.filter(r => r.isValid).map(r => r.data);
  const invalid = validated.filter(r => !r.isValid);

  // Insert all valid rows in one bulk operation
  const inserted = await dataInserter.insertAll(valid, batchId);

  fs.unlinkSync(req.file.path);
  res.json({ batchId, totalRows: rows.length, imported: inserted, rejected: invalid.length });
}
```

### Step 3: CSV Processor

Uses `csv-parse/sync` to parse entire content at once with `columns: true`, `trim: true`.

### Step 4: Row Validator

Validates required fields: `event_type` (string), `event_value` (numeric), `event_timestamp`
(valid date). Returns `{ isValid, data?, error? }`.

### Step 5: Data Inserter

Builds a single bulk INSERT with all rows, flat parameter array.
