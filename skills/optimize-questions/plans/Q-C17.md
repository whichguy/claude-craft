### Step 2: Notification Hook

`useNotifications.ts`:
```typescript
export function useNotifications() {
  const dispatch = useDispatch();
  const { items, unreadCount } = useSelector((s: RootState) => s.notifications);

  useEffect(() => {
    const ws = new WebSocket(`${WS_URL}/notifications`);
    ws.onmessage = (event) => {
      dispatch(addNotification(JSON.parse(event.data)));
    };

    const handleResize = () => {
      document.dispatchEvent(new CustomEvent('notification-reposition'));
    };
    window.addEventListener('resize', handleResize);

    const handleVisibility = () => {
      if (document.visibilityState === 'visible' && ws.readyState !== WebSocket.OPEN) {
        // reconnect
      }
    };
    document.addEventListener('visibilitychange', handleVisibility);

  }, [dispatch]);

  return { items, unreadCount };
}
```

### Step 3: Bell Component

Renders bell icon with unread badge. Toggles dropdown on click. Uses `useNotifications`
hook. Badge shows `99+` for counts over 99. CSS animation on `.has-unread`.

### Step 4: Dropdown Component

Lists recent 20 notifications. Click marks as read via `dispatch(markAsRead(n.id))`.
"Mark all read" button dispatches `markAllRead()`.

### Step 5: Header Integration

Import `NotificationBell`, render in `.header-actions` div alongside existing
`UserAvatar` and `SettingsLink`.
