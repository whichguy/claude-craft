# Plan: GAS Sidebar for Inventory Quick-Search

## Context

The warehouse team needs a sidebar in their Sheets inventory tracker for quick product
search by SKU or name. Replaces Ctrl+F on a 15,000-row sheet. Shows stock levels
and allows one-click navigation to the matching row.

## Current Architecture

- Sheets-bound GAS project, 15,000+ rows in "Inventory" sheet
- Columns: A=SKU, B=Name, C=Category, D=Stock, E=Location, F=LastUpdated
- No sidebar or custom UI

## Files to Change

| File | Change |
|------|--------|
| `src/sidebar.html` | New sidebar HTML/CSS/JS |
| `src/sidebarServer.gs` | Server-side search functions |
| `src/menu.gs` | Add menu item to open sidebar |

## Implementation Steps

### Step 1: Menu Registration

`onOpen()` adds "Inventory Tools > Quick Search" menu item.
`showSearchSidebar()` creates HtmlService sidebar, width 350.

### Step 2: Server-Side Search

`searchInventory(query)`:
- Gets all data from "Inventory" sheet
- Case-insensitive search on SKU (col A) and Name (col B)
- Returns max 50 matches with row, sku, name, category, stock, location

`navigateToRow(rowNumber)` — sets active range to the matched row.

### Step 3: Sidebar UI

Search input with 300ms debounce (minimum 2 chars). Results rendered as clickable
items showing SKU, name, stock level, and location. Low stock (<10) highlighted.

```javascript
searchInput.addEventListener('input', function() {
  clearTimeout(searchTimer);
  var query = this.value.trim();
  if (query.length < 2) { resultsDiv.textContent = ''; return; }
  resultsDiv.textContent = 'Searching...';
  searchTimer = setTimeout(function() {
    google.script.run
      .withSuccessHandler(displayResults)
      .searchInventory(query);
  }, 300);
});
```

Click handler calls `google.script.run.navigateToRow(item.row)`.

No `.withFailureHandler()` on the search or navigate calls — if the server function
throws (e.g., sheet renamed, permissions issue), the sidebar shows no feedback.
