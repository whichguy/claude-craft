# Plan: Async Data Pipeline — ETL from Multiple Sources

## Context

ETL pipeline pulling from four sources (S3, REST API, FTP, SQS webhook queue),
transforming, and loading into a PostgreSQL data warehouse. Runs every 15 minutes.

## Current Architecture

- Node.js data processing service, PostgreSQL warehouse
- Manual CSV imports, no automated pipeline

## Files to Change

| File | Change |
|------|--------|
| `src/pipeline/orchestrator.ts` | Main pipeline runner |
| `src/pipeline/extractors/s3Extractor.ts` | Pull CSV files from S3 |
| `src/pipeline/extractors/apiExtractor.ts` | Pull from partner REST API |
| `src/pipeline/extractors/ftpExtractor.ts` | Pull from legacy FTP |
| `src/pipeline/extractors/webhookExtractor.ts` | Drain SQS webhook queue |
| `src/pipeline/transformer.ts` | Normalize and validate records |
| `src/pipeline/loader.ts` | Bulk insert to warehouse |

## Implementation Steps

### Step 1: Four Extractors

Each returns `Promise<RawRecord[]>`:
- **S3**: list new files, download, parse CSV
- **API**: `axios.get(PARTNER_URL).then(r => r.data.records.map(normalize))`
- **FTP**: connect, list `/outbound`, download CSVs, parse
- **Webhook**: receive SQS messages, parse, delete processed messages

### Step 2: Orchestrator

```typescript
export async function runPipeline(): Promise<PipelineResult> {
  const [s3Data, apiData, ftpData, webhookData] = await Promise.all([
    extractFromS3(),
    extractFromApi(),
    extractFromFtp(),
    extractFromWebhookQueue()
  ]);
  const allRecords = [...s3Data, ...apiData, ...ftpData, ...webhookData];
  const transformed = transformer.process(allRecords);
  const loadResult = await loader.bulkInsert(transformed);
  return { extracted: allRecords.length, loaded: loadResult.inserted };
}
```

### Step 3: Transformer

Filters invalid records (missing timestamp or value), normalizes IDs, parses floats,
converts timestamps.

### Step 4: Loader

Bulk inserts in chunks of 500 using sequential Promise chain.

### Step 5: Scheduler

```typescript
cron.schedule('*/15 * * * *', () => {
  runPipeline().then(result => console.log('Pipeline complete:', result));
});
```

Note: The FTP extractor's `.then()` chain and webhook extractor's `.then()` chain
have no `.catch()` handlers. Only the S3 and API extractors include `.catch()`.
