# Plan: Batch Data Processor for Nightly Reconciliation

## Context

Nightly batch job reconciling transactions between our system and bank SFTP exports.
Downloads CSV files, matches ~50,000 records against the database, flags discrepancies,
and emails a report to the finance team.

## Current Architecture

- Node.js service with PostgreSQL (~2M transaction rows)
- Bank exports CSV to SFTP nightly at 2 AM
- Manual reconciliation via spreadsheet (4+ hours, error-prone)

## Files to Change

## Implementation Steps

### Step 1: SFTP Downloader

Connect via `ssh2-sftp-client`, list `/exports/daily/`, download all `.csv` files
to a temp directory.

### Step 2: CSV Parser

Parse each file using `csv-parse` with column headers. Extract reference, amount,
date, and description fields.

### Step 3: Record Matcher

```typescript
export async function matchRecords(bankRecords: BankRecord[]): Promise<MatchResult[]> {
  const pool = new Pool({ connectionString: DATABASE_URL });
  const results: MatchResult[] = [];

  for (const bankRecord of bankRecords) {
    const client = await pool.connect();
    const dbResult = await client.query(
      'SELECT id, amount, reference FROM transactions WHERE reference = $1',
      [bankRecord.bankRef]
    );
    if (dbResult.rows.length === 0) {
      results.push({ bankRef: bankRecord.bankRef, status: 'missing_ours', bankAmount: bankRecord.amount });
    } else {
      const match = Math.abs(dbResult.rows[0].amount - bankRecord.amount) < 0.01;
      results.push({
        bankRef: bankRecord.bankRef, ourRef: dbResult.rows[0].reference,
        bankAmount: bankRecord.amount, ourAmount: dbResult.rows[0].amount,
        status: match ? 'matched' : 'mismatch'
      });
    }
  }
  return results;
}
```

### Step 4: Report Generator

Summarizes matched/mismatched/missing counts. Stores results in `reconciliation_results`
table. Emails HTML report to finance team.

### Step 5: Orchestrator

```typescript
export async function runReconciliation(): Promise<void> {
  const tmpDir = path.join(os.tmpdir(), `recon-${Date.now()}`);
  fs.mkdirSync(tmpDir, { recursive: true });
  const files = await downloadBankFiles(tmpDir);
  const allRecords = [];
  for (const file of files) { allRecords.push(...await parseBankCsv(file)); }
  const results = await matchRecords(allRecords);
  await generateReport(results, new Date());
}
```

### Step 6: Scheduler

`cron.schedule('30 2 * * *', () => runReconciliation())`
